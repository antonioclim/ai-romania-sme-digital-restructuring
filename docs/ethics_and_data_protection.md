# Ethics and data-protection note

The public repository excludes raw LimeSurvey exports, IP addresses, exact timestamps and open-text fields. The retained dataset contains quantitative categorical variables, a synthetic case identifier and a coarse completion-month variable. Open-text material is not recommended for public release until a manual re-identification review is complete.

This note is not legal advice and does not claim GDPR compliance. It documents the data-minimisation decisions used in the reproducibility kit.
