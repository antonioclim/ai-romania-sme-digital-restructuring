# Methods and reproducibility

The kit reproduces survey-analysis outputs rather than storing static figures. All principal tables and figures are regenerated from the cleaned quantitative dataset. Multi-response items are represented as binary columns, while single-choice categorical items are tabulated by analytical scope.

The pipeline separates full completed responses from SME-only responses and treats large firms as a comparator group rather than as SMEs. This design is necessary because the full completed sample contains large firms and one case with missing firm-size information.
